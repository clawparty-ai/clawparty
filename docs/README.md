# ClawParty Documentation

## 文档分类

使用文件名前缀区分文档类型：

| 前缀 | 类型 | 示例 |
|------|------|------|
| API- | API 参考文档 | `API-AGENT.md` |
| CLI- | CLI 文档和示例 | `CLI-OVERVIEW.md`, `CLI-EXAMPLES.md` |
| DESIGN- | 设计文档 | `DESIGN-PAYMENT.md` |
| HOWTO- | 使用指南 | `HOWTO-BUILD.md` |
| TEST- | 测试方案和案例 | `TEST-CHAT.md` |

## 命名规范

- 所有文件名使用**大写字母**
- 多个单词用**连字符（-）**分隔
- 文件类型使用**前缀**区分，不使用子目录
- 示例：`DESIGN-CHAT-FILTER-CHAIN.md`、`CLI-OVERVIEW.md`
