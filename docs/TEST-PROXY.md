# Proxy App Test Plan

## 概述

Proxy App 是 ClawParty 的零信任代理应用，允许从任何位置访问设备。本文档记录了 Proxy App 的测试方案和测试案例。

## 测试范围

### 1. 基础功能测试
- CLI 帮助命令
- 代理配置
- 监听和转发端点

### 2. 监听端配置测试
- 设置监听地址
- 修改监听端口
- 关闭监听

### 3. 转发端配置测试
- 添加转发目标
- 移除转发目标
- 查看转发规则

### 4. 代理连接测试
- HTTP 代理
- HTTPS 代理
- SOCKS 代理（可选）

### 5. 域名和 IP 过滤测试
- 域名匹配
- IP 范围匹配
- 通配符支持

### 6. 多端点测试
- 多个转发端
- 多个监听端

---

## 测试案例

### TC-PX-001: CLI 帮助命令

**前置条件**: Proxy App 已启动

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 执行 `ztm proxy help` | 显示帮助信息 |
| 2 | 执行 `ztm proxy help config` | 显示 config 命令帮助 |
| 3 | 验证选项说明 | 显示 --set-listen, --add-target, --remove-target |

---

### TC-PX-002: 配置转发端

**前置条件**: 存在两个端点

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 执行 `ztm proxy config --add-target 0.0.0.0/0 '*'` | 配置成功 |

## 已知问题和注意事项

1. **端口要求**: 监听端需要可用的本地端口
2. **防火墙**: 需要确保端点间可以通信
3. **DNS 解析**: 使用转发端的 DNS 配置
4. **协议支持**: 支持 HTTP/HTTPS 代理
5. **目标匹配**: 支持域名、IP 范围和通配符
