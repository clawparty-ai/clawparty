# 用户管理测试方案

## 概述

用户管理是 ClawParty 的核心功能之一，包括用户注册、身份认证、状态管理和权限控制。本文档记录了用户管理功能的测试方案和测试案例。

## 测试范围

### 1. 用户注册测试
- 注册 API 可用性
- 注册参数验证
- 证书签发
- Permit 生成

### 2. 用户状态测试
- 状态流转
- 状态查询
- 状态变更

### 3. 用户封禁测试
- 封禁用户
- 解除封禁
- 封禁效果

### 4. 身份认证测试
- 证书验证
- 连接建立
- 权限检查

### 5. 审计日志测试
- API 日志
- 用户事件日志
- 日志查询

### 6. CLI 命令测试
- 加入 Mesh
- 配置管理
- 用户操作

## 已知问题和注意事项

1. **注册 API 安全**: 注册端口为明文 HTTP，需要网络安全措施
2. **证书有效期**: 默认 365 天，过期需重新注册
3. **封禁时间**: 封禁有时间戳和过期时间
4. **用户记录**: 删除用户记录才能重新注册
5. **日志保留**: 日志表会持续增长，需要定期清理
